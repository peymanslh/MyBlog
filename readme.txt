MyBlog WordPress theme is a fully responsive theme that looks great on any devices including iPad, iPhone or any other mobile device. Some other great features are responsive layout, simplicity and minimalism.
Support: https://github.com/peymanslh/MyBlog/issues

== Installation ==
Manual installation:

1. Upload the `MyBlog` folder to the `/wp-content/themes/` directory

Activiation and Use

1. Activate the Theme through the 'Themes' menu in WordPress
2. See Appearance -> Theme Options to change theme specific options

== Custom Logo ==
This theme supports custom logo for the header part.
You can change or remove it from Appearance -> Theme Options.
